<!-- 40-RequiredProperties -->  
Un concepto SKOS puede considerarse una idea o noción, una unidad de pensamiento. Sin embargo, lo que constituye una unidad de pensamiento es subjetivo, y esta definición pretende ser sugerente, más que restrictiva.  
La noción de concepto SKOS es útil para describir la estructura conceptual o intelectual de un sistema de organización del conocimiento, y para referirse a ideas o significados específicos establecidos dentro de un KOS.  
Nótese que, dado que SKOS está diseñado para ser un vehículo de representación de KOS semiformales, como tesauros y esquemas de clasificación, se ha incorporado cierta flexibilidad a la definición formal de esta clase.  
Véase [https://www.w3.org/TR/skos-reference/#ref-SKOS-PRIMER] para más ejemplos de identificación y descripción de conceptos SKOS.
