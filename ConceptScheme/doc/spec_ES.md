<!-- 40-RequiredProperties -->  
Un esquema conceptual SKOS puede considerarse como una agregación de uno o más conceptos SKOS. Las relaciones semánticas (enlaces) entre esos conceptos también pueden considerarse parte de un esquema conceptual. No obstante, esta definición pretende ser sugerente y no restrictiva, y existe cierta flexibilidad en el modelo formal de datos que se expone a continuación.  
La noción de esquema conceptual es útil cuando se trata de datos de una fuente desconocida, y cuando se trata de datos que describen dos o más sistemas de organización del conocimiento diferentes.  
Véase [https://www.w3.org/TR/skos-reference/#ref-SKOS-PRIMER] para más ejemplos de identificación y descripción de esquemas conceptuales.
